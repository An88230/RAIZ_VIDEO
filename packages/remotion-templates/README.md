# RAIZ Remotion Templates

This package is reserved for RAIZ-owned Remotion templates.

Phase 1 only defines the Arabic RTL rules and folder shape. It does not render video and does not import upstream Remotion examples from `vendor/`.

Starter template target:

```text
arabic-rtl/
  README.md
```
