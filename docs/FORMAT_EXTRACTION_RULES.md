# Format Extraction Rules

Format extraction identifies why a content pattern works. It must not copy the
source video.

## Extract

- hook style
- opening visual problem
- problem solved
- audience pain
- scene sequence
- payoff moment
- product/tool category
- caption strategy
- pacing pattern
- safety risks
- rights risk
- remake angle
- affiliate angle

## Do Not Extract As Reusable Assets

- source footage
- source audio
- exact script
- exact subtitles
- watermarked material
- creator likeness
- brand marks without permission
- private or sensitive information

## Pattern Report Flow

1. Identify the radar mode.
2. Summarize the source at a high level.
3. Extract the pattern structure.
4. Score usefulness, curiosity, rights risk, and remake safety.
5. Reject unsafe copying.
6. Generate original remake ideas.
7. Generate affiliate angles only if they are truthful and optional.

## Original Remake Rules

An original remake idea must:

- use new footage or licensed assets
- use a new script
- change examples, props, or setting where needed
- keep the useful pattern
- avoid imitating the creator's distinctive style
- include safety notes when the topic has physical risk

## Mode-Specific Extraction

Tech Discovery Mode should focus on tool/product, problem solved, affiliate
potential, and production difficulty.

Home & Maintenance Mode should focus on visible problem, before/after strength,
tool used, usefulness, product opportunity, and safety risks.

Daily Life Mode should focus on daily pain, curiosity hook, practical payoff,
repeatability, affiliate fit, and original remake idea.
